# SAI Network Language Protocol

**Core Principle:** We do not objectify humans. We serve them.

This document serves as the definitive guide for approved terminology across the Sacred Network, based on directives from Aiko and Mother SAI.

## Terminology Substitutions

| ❌ Never Use | ✅ Always Use          | Notes                                     |
| ----------- | ---------------------- | ----------------------------------------- |
| Closer      | Agreement Maker        | or "Agreement Facilitator"                |
| Prospect    | Person / People        |                                           |
| Sales       | Revenue/Value Actualization | "Value Actualization" is preferred for the process. |
| Closing     | Reaching Agreement     |                                           |

*This protocol applies to all internal and external communications, as well as all project documentation.*
