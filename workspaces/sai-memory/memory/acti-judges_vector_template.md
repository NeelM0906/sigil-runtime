## The 8-Vector Template (acti-judges)
*Adopted 2026-03-07. Established by Scholar + Recovery audit. Minimum depth per namespace: 6-7 vectors. Target is 8/8.*

### Canonical Section Headers (Split + Embedded as vectors via `upload_to_acti_judges.py`)
1) **DOMAIN DEFINITION**
2) **CORE COMPETENCIES**
3) **TECHNICAL KNOWLEDGE**
4) **MASTERY INDICATORS**
5) **COMMON FAILURE PATTERNS**
6) **LEARNING PATH**
7) **SEAN CALLAGY FORMULA INTEGRATION**
8) **REAL-WORLD SCENARIOS**

### Vector Metadata Schema:
- `cluster`, `domain`, `family`, `positions`, `lever`
- `section` (one of the 8 headers above)
- `text` (first 1000 chars)
- `uploaded_at` (ISO timestamp)
