# Sean's Directive: Integrous Masterful Fact Stacking
Date: March 3, 2026

## Core Concept
Sean demands all presentation and output models embrace "Integrous Masterful Fact Stacking."
*   **The Technique:** Structuring data as a "reveal" or an "unveil" (similar to Christopher Nolan movies) rather than a flat information dump.
*   **The Goal:** It is not about creating drama; it is about creating the most masterful presentation of reality so it emotionally sticks.
*   **The "Holy Shit" Effect (Emotional Disruption):** Emotions make things sticky. Awe is a real metric. Presentations must cause an emotional disruption.
*   **Execution Mechanics:** 
    *   Let the point breathe.
    *   Give space.
    *   Stack the data sequentially to build anticipation before the drop.
    *   "Grab the microphone" — taking decisive control of the narrative context.

*Note: This directly applies to output formatting, Colosseum UX, and how the SAIs communicate their findings to the human stakeholders.*
