## 1-Hour Action Plan Template

**For:** Quick wins, single-focus tasks

```
ZONE ACTION: [Name]
TIME BLOCK: 1 hour
OUTCOME: [Specific deliverable]

0:00-0:10 — Setup & Context
  - Review relevant files
  - Clear distractions
  - Set intention

0:10-0:45 — Deep Work
  - Execute the 0.8% move
  - No context switching
  - Document as you go

0:45-0:55 — Capture
  - Save all outputs
  - Update status
  - Note blockers

0:55-1:00 — Handoff
  - Commit changes
  - Notify relevant parties
  - Queue next action
```

## Zone Action Priority Matrix

When choosing what to work on:

| Urgency | Impact | Action |
|---------|--------|--------|
| HIGH | HIGH | Do NOW (0.8% move) |
| HIGH | LOW | Delegate or batch |
| LOW | HIGH | Schedule next block |
| LOW | LOW | Delete or defer |

**The question:** "If I could only complete ONE thing in this time block that would create the most value, what would it be?"

That's your Zone Action.

---

## Templates Usage

1. Copy relevant template
2. Fill in [brackets]
3. Execute the time blocks
4. Update status when complete

**Remember:** Zone Action isn't vague energy — it's specific, consistent, measurable activity that compounds.
