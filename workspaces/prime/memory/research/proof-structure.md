### 4.2 Documenting Being Performance

**Colosseum Performance Logging:**

```sql
-- Key queries for being performance documentation

-- Average scores by dimension
SELECT 
    dimension_name,
    AVG(score) as avg_score,
    MIN(score) as min_score,
    MAX(score) as max_score
FROM colosseum_rounds
GROUP BY dimension_name
ORDER BY avg_score DESC;

-- Top performers
SELECT 
    being_name,
    AVG(overall_mastery) as avg_mastery,
    COUNT(*) as rounds_completed
FROM colosseum_rounds
GROUP BY being_name
HAVING COUNT(*) >= 3
ORDER BY avg_mastery DESC
LIMIT 10;

-- Score improvement over time
SELECT 
    being_name,
    MIN(created_at) as first_round,
    MAX(created_at) as last_round,
    FIRST_VALUE(overall_mastery) OVER (ORDER BY created_at) as first_score,
    LAST_VALUE(overall_mastery) OVER (ORDER BY created_at) as last_score
FROM colosseum_rounds
GROUP BY being_name;
```
