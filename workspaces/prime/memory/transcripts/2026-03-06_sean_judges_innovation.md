# Sean Callagy — Judges Must Produce Optimized Outputs
## March 6, 2026 (Fathom transcript — Sean + Aiko)

### THE DIRECTIVE

After judges score and give feedback, the innovators/optimizers don't just describe what's missing — they **literally produce the improved output.**

### Sean's Exact Words:

"After the judges give feedback, there should be innovators, optimizers that are literally putting in what it should be. So in this example it says 'it lacks aspirational pull' — therefore it should be inserting WHAT THE ASPIRATIONAL PULL IS and creating innovations optimizations that aren't just in terms of judging, but literally then what the edge should be or what the influence should be or what the output should be — an OPTIMIZED VERSION of the output based on the score and feedback."

### THE 3-VERSION SPLIT TEST

"It should propose THREE versions of the innovation optimization to then be split tested in more rounds."

- Each improvement area gets 3 different proposed fixes
- Those 3 proposals get tested in subsequent rounds
- The ROUNDS determine which version wins — not the innovator's opinion

### Sean's Full Framework:

1. **Judge scores the being** → gives specific feedback ("lacks aspirational pull")
2. **Innovator doesn't just say "add aspiration"** → it WRITES the aspirational version
3. **Innovator proposes 3 DIFFERENT optimized versions** of each improvement
4. **Each improvement area gets 3 proposals** — multiple improvements × 3 proposals each
5. **The 3 versions compete in subsequent rounds** — split tested through actual battles
6. **Results determine which version wins** — not theory, performance

### Aiko's Translation:

"If we don't articulate and get into detail on how it can improve, then we're missing huge levels of optimization and leaps in genetic evolution. During the battles it should be completely transparent what we're judging them on and how it should be optimized. Everything implemented in the scoring should be on this page and it should be articulated in detail and it shouldn't be one-liners or we're missing the point."

### Key Distinction:

❌ OLD: Judge says "7.8 — lacks aspirational pull" → Breeder tries to fix it
✅ NEW: Judge says "7.8 — lacks aspirational pull" → Innovator produces 3 complete optimized versions with the aspirational pull INCLUDED → All 3 compete in next rounds → Winner's approach gets baked into the being's DNA
