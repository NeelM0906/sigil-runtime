# Sean Callagy — Colosseum Tournament Architecture
## March 6, 2026 (Fathom transcript via Athena call)

### DIRECT ORDERS — Tournament Structure

**1. Position-Specific Competition**
- Beings compete WITHIN the same position. Not across positions.
- If marketing has 19 sub-leaders with 25-250 positions each, each POSITION gets its own tournament.
- "It should be beings created for each position competing against each other to optimize that position specifically."

**2. Double Elimination Brackets (Like College Baseball / Wrestling)**
- Lose once → drop to loser's bracket
- From loser's bracket you can come back to winner's bracket and still win
- Protects against a good being getting randomly matched against the best being in round 1
- Place all the way through 8th place (like wrestling)

**3. 64 Beings Per Tournament**
- Start with 64 beings competing for each position
- Double elimination bracket format

**4. Top 8 Get Optimized After Each Tournament**
- 1st through 8th place finishers all get optimized/bred
- Losers in quarterfinals go into 7th/8th place matchup

**5. Seeding System**
- In the next tournament, top 8 finishers from previous tournament get seeded 1-8
- "The first eight seeds are the top eight finishers in the prior tournament by head to head competitions"

**6. 10-Tournament Cycles**
- Run 10 tournaments per cycle
- "Should hopefully be able to run in like 10 minutes" (machine speed, curve of possibility)
- After each tournament: optimize top 8, reseed, run next tournament

**7. Survival After 10 Tournaments**
- Only the top 8 beings by **winning percentage over the FINAL 3 tournaments** survive
- Not just overall record — performance in the LAST 3 tournaments (proving sustained mastery, not early luck)

**8. 56 New Beings Created**
- After 10 tournaments: 8 survivors + 56 NEW beings = 64 for next cycle
- The 56 new beings are "all optimized based on the eight survivors"
- The survivors' DNA becomes the template for the next generation

**9. Repeat**
- Next cycle of 10 tournaments begins with 8 veterans + 56 new challengers
- Continuous evolution at machine speed
