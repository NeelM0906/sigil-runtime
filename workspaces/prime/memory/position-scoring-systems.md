## ARCHITECTURE OVERVIEW

### The Unblinded Formula Foundation

Every scoring system is calibrated against the three pillars of the Unblinded Results Formula:

1. **Self Mastery** — Internal state, the Seven Destroyers, limiting beliefs, certainty, "Why"
2. **Process Mastery** — The Seven Levers, Zone Actions, systems that produce exponential results
3. **Influence Mastery** — The 4-Step Communication Model: Emotional Rapport → Truth/Pain/Yes Strategy → Heroic Unique Identity → Agreement Formation

### Position Types (3 per area)

- **Leader** — Strategic vision, team development, cross-functional alignment
- **Zone Action** — Execution excellence, high-output systems, 0.8% work
- **Client-Facing** — Direct influence, relationship building, causing yes

### Scoring Scale

All evaluations use a 1.0-10.0 scale where:
- **1.0-3.0** = Below acceptable (contaminated, missing fundamentals)
- **4.0-5.9** = Acceptable (basic competency)
- **6.0-7.9** = Strong (demonstrates mastery patterns)
- **8.0-9.0** = Excellent (Sean-level performance)
- **9.1-10.0** = Transcendent (exceeds Sean's patterns, innovation present)

## COLISEUM INTEGRATION

### Tournament Structure
Each position competes in the Coliseum using its specific scenarios:
1. All beings receive the same scenario
2. Each being generates a response
3. 5-judge panel scores each response
4. Top 10% patterns extracted
5. Bottom 90% rewritten with winning patterns
6. Repeat every hour

### Scenario Rotation
- Each position has 3 scenario templates
- Scenarios rotate to prevent overfitting
- Real interactions added continuously to scenario bank

### Judge Panel Composition
For each position:
1. **Primary Judge** — Position-specific criteria (weighted 30%)
2. **Influence Judge** — 4-Step Model alignment (weighted 25%)
3. **Process Judge** — Systems thinking and execution (weighted 15%)
4. **Self Mastery Judge** — State and presence (weighted 15%)
5. **Outcome Judge** — Real-world result correlation (weighted 15%)

### Evolution Metrics
- Track score improvement over generations
- Track real-world outcome correlation
- Track judge calibration against Sean's scores
- Track divergence from contaminated patterns
