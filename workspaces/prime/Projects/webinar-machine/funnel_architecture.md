# Funnel Architecture — Webinar Marketing Machine

## Overview

This document details the complete funnel architecture from initial awareness through post-webinar conversion and follow-up. The focus is on **CONVERSION**, not just influence.

---

## The 7-Stage Funnel

```
┌─────────────────────────────────────────────────────────────────────┐
│                         AWARENESS                                    │
│                   (Cold Traffic Entry)                               │
├─────────────────────────────────────────────────────────────────────┤
│                         CAPTURE                                      │
│               (Landing Page + Registration)                          │
├─────────────────────────────────────────────────────────────────────┤
│                         NURTURE                                      │
│                (Pre-Webinar Email Sequence)                          │
├─────────────────────────────────────────────────────────────────────┤
│                        ATTENDANCE                                    │
│                   (Live Webinar Event)                               │
├─────────────────────────────────────────────────────────────────────┤
│                        CONVERSION                                    │
│               (Offer Presentation + CTA)                             │
├─────────────────────────────────────────────────────────────────────┤
│                        CLOSE                                         │
│           (Post-Webinar Sequence + Sales)                            │
├─────────────────────────────────────────────────────────────────────┤
│                       EXPANSION                                      │
│             (Upsell + Referral + Retention)                          │
└─────────────────────────────────────────────────────────────────────┘
```

## Stage 1: AWARENESS

### Goal
Generate qualified traffic to registration pages.

### Traffic Sources

| Channel | ACT-I | Unblinded | Callagy Recovery |
|---------|-------|-----------|------------------|
| Facebook/Instagram Ads | ✓ | ✓ | ✓ |
| LinkedIn Ads | ✓ (Primary) | ✓ | ✓ |
| Google Search | ✓ | ✓ | ✓ (Primary) |
| YouTube Pre-roll | ✓ | ✓ | - |
| Email List | ✓ | ✓ | ✓ |
| Podcast Mentions | ✓ | ✓ | - |
| Partner Affiliates | ✓ | ✓ | ✓ |

### Ad Creative Framework

**Hook Types (Split Test):**
1. **Pain Agitation:** "Tired of [problem]?"
2. **Curiosity Gap:** "The [X] secret top [audience] know..."
3. **Social Proof:** "[Number] [audience] already [achieved result]"
4. **Controversy:** "Why [conventional wisdom] is wrong"
5. **FOMO:** "The [industry] shift happening right now..."

**Creative Formats:**
- Video (15s, 30s, 60s variants)
- Static image + copy
- Carousel (problem → solution → CTA)
- Native/editorial style

### Targeting Strategy

**ACT-I:**
- Job titles: Attorney, Partner, Legal Counsel, Litigation
- Interests: Legal tech, Case management, Trial preparation
- Behaviors: Legal software users, Conference attendees

**Unblinded:**
- Job titles: Executive, Founder, CEO, Director, VP
- Interests: Personal development, Leadership, Executive coaching
- Behaviors: Business book buyers, Podcast listeners

**Callagy Recovery:**
- Job titles: PT, Physical Therapist, Chiropractor, Practice Owner
- Interests: Medical billing, Practice management, Healthcare business
- Behaviors: Medical software users, Association members

```
│          WEBINAR PROMISE               │

│  • Privacy/spam assurance              │

```

### Registration Form Fields

**Essential (Required):**
- First Name
- Email Address

**Optional (Segment/Qualify):**
- Phone Number (SMS reminders)
- Company/Practice Name
- Role/Title
- Biggest Challenge (dropdown)

**Split Test: Form Length**
- Variant A: Name + Email only (higher conversion, lower quality)
- Variant B: Name + Email + Phone (medium conversion, SMS enabled)
- Variant C: Full form (lower conversion, higher quality)

### Thank You Page Elements

1. **Confirmation:** "You're registered! Here's what happens next..."
2. **Calendar Add:** Google/iCal/Outlook buttons
3. **Share Incentive:** "Invite a colleague, unlock bonus content"
4. **Ascension Offer:** "Can't wait? Book a strategy call now" (split test)
5. **Community Invite:** Facebook group or Slack channel

## Stage 3: NURTURE

### Pre-Webinar Email Sequence

**Email 1: Registration Confirmation (Immediate)**
- Subject: "You're in! [Webinar Name] confirmed"
- Content: Date/time, add to calendar, what to expect
- CTA: Add to calendar

**Email 2: Value Building (Day -5)**
- Subject: "Before the webinar: [Valuable insight]"
- Content: Pre-frame key concept, build anticipation
- CTA: Prepare your questions

**Email 3: Social Proof (Day -3)**
- Subject: "[Name], here's what [past attendee] said..."
- Content: Testimonial, case study snippet
- CTA: Invite a colleague

**Email 4: Agenda Preview (Day -1)**
- Subject: "Tomorrow: Here's exactly what we'll cover"
- Content: Detailed agenda, what to have ready
- CTA: Set your reminder

**Email 5: Day-Of Reminder #1 (Morning)**
- Subject: "TODAY: [Webinar Name] starts at [time]"
- Content: Quick benefits reminder, link
- CTA: Join now (if waiting room open)

**Email 6: Day-Of Reminder #2 (1 hour before)**
- Subject: "Starting in 60 minutes!"
- Content: Direct join link, urgency
- CTA: Join the webinar

**Email 7: Day-Of Reminder #3 (15 minutes before)**
- Subject: "We're starting! Join now →"
- Content: Just the link
- CTA: JOIN NOW

### SMS Sequence (If Phone Collected)

- Day -1: "Reminder: [Webinar] tomorrow at [time]. Can't wait to see you!"
- Day 0 (2h before): "[Webinar] starts in 2 hours. Reply YES to get the join link"
- Day 0 (15m before): "We're starting! Join now: [link]"
- Day 0 (during, if no-show): "You're missing it! Join now: [link]"

## Stage 4: ATTENDANCE

### Live Webinar Flow

**Technical Setup:**
- Platform: Demio or WebinarJam
- Backup: Zoom with webinar license
- Recording: Always on
- Chat: Moderated, engagement prompts

**Engagement Tactics:**

| Minute | Tactic | Purpose |
|--------|--------|---------|
| 0-5 | Welcome poll | Engagement, segment data |
| 10 | First chat prompt | Keep attention |
| 20 | Quick poll | Re-engagement |
| 30 | Case study intro | Proof, attention reset |
| 45 | "Drop a 🔥" moment | Peak engagement |
| 60 | Pre-offer poll | Prime for offer |
| 70 | Q&A | Objection handling |

**Retention Hooks:**
- "In 10 minutes, I'll reveal [specific thing]..."
- "Stay until the end for [exclusive bonus]..."
- "The most important part is coming up..."

## Stage 5: CONVERSION

### The Offer Presentation (10-15 minutes)

**Structure:**
1. **Transition:** "Now, some of you are wondering how to implement this..."
2. **Introduce Program:** Name, promise, delivery method
3. **Modules/Components:** What's included (visual stack)
4. **Results Preview:** What transformation looks like
5. **Bonuses:** Time-limited, webinar-only
6. **Investment:** Price reveal (anchored high first)
7. **Guarantee:** Risk reversal
8. **Scarcity:** Limited spots, deadline
9. **CTA:** Clear next step

### Offer Stack Template

```
┌────────────────────────────────────────┐
│         [PROGRAM NAME]                 │
│         Value: $X,XXX                  │
├────────────────────────────────────────┤
│    + Bonus 1: [Thing]                  │
│      Value: $X,XXX                     │
├────────────────────────────────────────┤
│    + Bonus 2: [Thing]                  │
│      Value: $XXX                       │
├────────────────────────────────────────┤
│    + Bonus 3: [Thing] (24hr only)      │
│      Value: $X,XXX                     │
├────────────────────────────────────────┤
│    + GUARANTEE: [Risk Reversal]        │
├────────────────────────────────────────┤
│    TOTAL VALUE: $XX,XXX                │
│                                        │
│    YOUR INVESTMENT TODAY:              │
│    $X,XXX (or $XXX/month)              │
└────────────────────────────────────────┘
```

### CTA Options (Split Test)

- **Option A:** Direct purchase link (low-ticket, <$2k)
- **Option B:** Application form (high-ticket, >$2k)
- **Option C:** "Schedule a call" (premium, >$10k)

## Stage 6: CLOSE

### Post-Webinar Email Sequence (Non-Buyers)

**Email 1: Replay (Hour +1)**
- Subject: "Replay: [Webinar Name] + Special Offer Inside"
- Content: Replay link, offer recap, deadline reminder
- CTA: Watch replay / Claim offer

**Email 2: Case Study Deep Dive (Day +1)**
- Subject: "[Name], see how [Client] achieved [result]"
- Content: Full case study, connect to offer
- CTA: Get similar results

**Email 3: FAQ/Objection Handling (Day +2)**
- Subject: "Your questions answered about [Program]"
- Content: Common questions, overcome objections
- CTA: Still have questions? Let's talk

**Email 4: Scarcity Reminder (Day +3)**
- Subject: "[Bonus] expires in 24 hours"
- Content: What they'll miss, deadline
- CTA: Don't miss out

**Email 5: Final Call (Day +5)**
- Subject: "Last chance: [Offer] closes tonight"
- Content: Summary, urgency, testimonial
- CTA: Final offer

**Email 6: Closed + Next Opportunity (Day +7)**
- Subject: "We noticed you didn't join..."
- Content: Empathy, next cohort waitlist, downsell option
- CTA: Join waitlist / Check out [alternative]

### Sales Team Follow-Up (High-Ticket)

| Day | Action | Channel |
|-----|--------|---------|
| 0 | Immediate outreach if application submitted | Phone |
| +1 | Follow-up if no response | Email + SMS |
| +2 | Second call attempt | Phone |
| +3 | Objection-handling email | Email |
| +5 | Final call attempt | Phone |
| +7 | Long-term nurture sequence | Email |

## Stage 7: EXPANSION

### Upsell Path

```
Entry Offer ($2-5k)
       ↓
Implementation Coaching ($5-10k)
       ↓
Done-With-You Program ($10-25k)
       ↓
Done-For-You Service ($25k+)
       ↓
Strategic Partnership (Custom)
```

### Referral Program

- **Trigger:** Post-purchase success milestone
- **Incentive:** $500 credit or bonus for each referral who registers
- **Tracking:** Unique referral links, automated attribution

### Retention Elements

- **Community:** Private Facebook group, Slack, or Circle
- **Check-ins:** Automated progress emails
- **Office Hours:** Weekly Q&A calls
- **Wins Board:** Celebrate client successes
- **Renewal:** Annual membership or maintenance option
