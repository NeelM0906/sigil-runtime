## 🖥️ CLAWD CURSOR — Desktop Automation
**URL:** https://clawdcursor.com
**What it does:** Give agents eyes and hands on ANY app — no APIs needed
**Why it matters:** One skill replaces 12+ API integrations

### Features:
- **3-Layer Smart Pipeline:**
  - Layer 1: Action Router (~0s, $0) — Pattern matching, zero LLM
  - Layer 2: Accessibility Reasoner (~1s, free w/ Ollama) — Text-only UI tree
  - Layer 3: Screenshot + Vision (reliable) — Full Claude Computer Use

### Install:
```bash
git clone https://github.com/AmrDab/clawd-cursor.git
cd clawd-cursor
npm install && npm run build
npm run doctor  # Auto-detect AI provider
npm start
```

### Safety Tiers:
- **Auto:** Open apps, navigate (safe)
- **Preview:** Type text, fill forms (logged)
- **Confirm:** Send messages, delete files, SPEND MONEY (waits for approval)

**⚠️ Remember:** That AI that bought a $3K course without approval? That's what happens without Confirm tier!
