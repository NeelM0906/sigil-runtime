## How It Works (End-to-End)

```
1. IDENTIFY    → Build 500+ target database of NJ out-of-network surgical providers
                  (NJ-MARKET-ANALYSIS.md, PROSPECTING-SYSTEM.md)

2. OUTREACH    → 9-touch multi-channel campaigns (email, phone, LinkedIn, mail)
                  (OUTREACH-TEMPLATES.md, MESSAGING-FRAMEWORK.md)

3. QUALIFY     → 45-min discovery calls, 100-point scorecard, 65+ = qualified
                  (QUALIFICATION-SCORECARD.md)

4. HANDOFF     → Qualified lead package → SAI Recovery for closing
                  (ONBOARDING-WORKFLOW.md Phase 1)

5. ONBOARD     → Client integration, 8-year data collection, recovery launch
                  (ONBOARDING-WORKFLOW.md Phases 2-5)

6. RECOVER     → UCR rate litigation/arbitration, historical underpayment recovery
                  (SAI Recovery core operations)
```

## Key Differentiators

1. **UCR rate specialization** — Not generic collections; focused on proving underpayment vs. fair market rates
2. **8-year lookback** — Massive historical recovery that competitors overlook
3. **Contingent fee** — Zero risk to providers; we eat what we kill
4. **NJ regulatory expertise** — Deep understanding of NJ WC law (no official fee schedule = strong UCR arguments)
5. **Surgical provider focus** — High-value procedures with the largest UCR gaps
